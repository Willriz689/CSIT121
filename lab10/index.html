Lab 10: Web Standards

This lab explains the UI Events KeyboardEvent code Values web standard. This is a standardized way of using keyboard input on a web page to work with the web page, or let the user interact with the web page.

What will you learn?

  • What is it? - Explains what the standard is in more detail
  • How is it Used? - This page shows how the web standard is used in practical application, with examples
  • Why Important? - Explains why this web standard matters for developers and users

Quick Info

  • It helps make keyboards work the same in all browsers
  • It makes websites more accessible
  • It helps developers write better code
  • All modern browsers support it
