Use Case: Keyboard Shortcuts

The second example shows a practical use on how this standard can be used. Developers use this standard to standardize the keys used on the keyboard to create shortcuts on their site. This can be for things such as saving a document on OneDrive web version, or saving a game on a online gaming site. Use the shortcuts given on the second example to see how this can work.

Use Case: Accessibility

There are many web users who are not able to use a mouse, whether it be disability, the computer they are using, or else. To make a website more accessible to these people, developers can program keys to do certain things on their site that you normally would do with a mouse. In this example, you can see how you can use the keyboard only to submit a form that would otherwise need a mouse to navigate, and the key being used is standardized across all keyboards and browsers. Use the “Enter” Key on your keyboard to traverse this.

Use Case: Games and Interactive Movement

Many websites have games that need keyboard controls. This standard makes it easy for developers to detect when a player presses keys to move a character around the screen or control the game. Without this standard, different browsers and keyboards would report key presses differently.

Conclusion

There are many ways that this web standard is important. Almost every major website uses keypresses to control a piece of their site. With this web standard, it creates a way to make how these inputs are taken universal. This allows more accessibility, and user friendliness to every site no matter the user's language, location, or needs.
