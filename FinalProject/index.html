Introduction

What is this project?

This is the final project for CCBC's CSIT 121 "Web Standards" class. The purpose of this project is to make a website that contains all labs created throughout the semester, and turn it into one web page. As you go through this site, you can click the "Toggle Style" on the bottom right of each page to switch to the original styling of the site. As you progress through the labs, note the difference in styling. With each lab, you will see what the class focused on week by week in learning web standards and development.
