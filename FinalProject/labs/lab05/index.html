Labs 1 & 2

While I have learned a lot already, I believe that my favorite thing that I have learned is how to use GitHub. I previously had limited experience with github, and only knew the basics of uploading a file. I now know how to share a repository, and how to host a site through GitHub! I have also learned a lot more tags for HTML than I previously knew, as well as more resources to research from.

Recapped: What I have learned (Labs 1 & 2)

  • How to host a site on GitHub
  • How to upload files to github through the GUI
  • More HTML tags for organization and structure

Lab 3

In this project I have learned a lot about using CSS to adjust the positioning and style of the elements better. This includes changing the color of links, adding borders to the elements, and changing the color inside of the borders. I have also learned how to connect a style document to a HTML document.

Recapped: What I have learned (Lab 3)

  • Using CSS to change link colors
  • How to create borders in CSS
  • Using an external style doc

Lesson 3: Useful Sites!

  1. Mozilla Developer Network Web Docs, Challenge: Styling a Biography Page While this is a class challenge, I used it as a reference in the solution code for support in this lab.
  2. W3 Schools: CSS Tutorial This guide is good for the basics of CSS, with styling and coloring elements.
  3. W3 Schools: CSS Selectors This website does a very good job at explaining selectors.

Lab 5

I learned more from this lesson than any other so far taught. I had a lot of trouble when starting this project, as I did not completely understand how the grid and flexbox worked. This project challenged me throughout, but overall helped me understand a lot of the basics better as well through it. I learned a lot about the grid and flexbox styling in CSS in this project. I also learned a lot about how the media queries work, and how you can change your site depending on the screen size. I thoroughly enjoyed this project even though it challenged me.

Recapped: What I have learned (Lab 5)

  • Using flexbox to style a website
  • How to further edit a navbar
  • How to make a site change based on screen size

Lesson 5: Useful Sites!

  1. GeeksForGeeks: Introduction to CSS Flexbox This was very useful as a reference to flexbox principles in this project.
  2. CSS Tricks: A guide to flexbox This was also used as a general guide to flexbox. Used to reference the principals and get examples.
  3. MDN Docs: Controlling Ratios of Flex Items Along the Main Axis This site was heavily used for how to control the size and shape of things using flexbox.
  4. Malven: Flexbox Yet another great guide, this one having more of a cheatsheet style to it, on how to use flexbox items
  5. W3Schools: Responsive Flexbox A general guide on flexbox, used for the code snippet examples for guiding my own coding.
  6. W3Schools: Horizonal Navigation Bars Used as a guide to create the horizontal navigation bar.
  7. MDN Docs: flex-flow Used as a guide on how to use the flex-flow property.
