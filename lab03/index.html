Learning HTML & CSS

Labs 1 & 2

While I have learned a lot already, I believe that my favorite thing that I have learned is how to use GitHub. I previously had limited experience with github, and only knew the basics of uploading a file. I now know how to share a repository, and how to host a site through GitHub! I have also learned a lot more tags for HTML than I previously knew, as well as more resources to research from.

Recapped: What I have learned (Labs 1 & 2)

  • How to host a site on GitHub
  • How to upload files to github through the GUI
  • More HTML tags for organization and structure

Lesson 2: Useful Sites!

  1. W3 Schools: HTML Basic ExamplesThis has been my go to guide the past two weeks when I am in need of assistance with basic tags.
  2. FreeCodeCamp: HTML Cheat Sheet This is a good go-to guide for quick and easy explanations as well as remembering how to set up the document initially.

Lab 3

In this project I have learned a lot about using CSS to adjust the positioning and style of the elements better. This includes changing the color of links, adding borders to the elements, and changing the color inside of the borders. I have also learned how to connect a style document to a HTML document.

Recapped: What I have learned (Lab 3)

  • Using CSS to change link colors
  • How to create borders in CSS
  • Using an external style doc

Lesson 3: Useful Sites!

  1. Mozilla Developer Network Web Docs, Challenge: Styling a Biography Page While this is a class challenge, I used it as a reference in the solution code for support in this lab.
  2. W3 Schools: CSS Tutorial This guide is good for the basics of CSS, with styling and coloring elements.
  3. W3 Schools: CSS Selectors This website does a very good job at explaining selectors.
