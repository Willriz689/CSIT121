Description

This photo is from the NGINX site. NGINX is an open source web server software, that allows you to host your own websites online yourself, instead of with a hosting service.

File Type Info

SVG is short for Scalable Vector Graphic file. This is a standard graphics file used to render 2D images online. These are used for graphics at any scale and are optimized for search engines. They do not contain pixels. This is best for user interface elements, icons, and diagrams.

Why did I choose this image?

I picked this particular image as I love open source software as a concept, and have been interested in using NGINX for web services for a while. Since this is a web standards class, I believed it fitting for the class type.
