Description

Cape Lookout is a lighthouse on the southern outer banks in North Carolina. The lighthouse that we see standing today was built in 1859, but it was the second lighthouse to be at this spot. The first lighthouse at this spot, also named Cape Lookout, was completed and lit in 1812. The first lighthouse was too short to light Lookout Shoals, part of the more dangerous areas of the Graveyard of the Atlantic.

File Type Info

JPG stands for Joint Photographic Experts Group; it is one of the most common formats for photographs. JPG photos can use up to 16.7 million colors, and the file can be compressed, as it uses lossy compression. This file type cannot be animated, and cannot be transparent.

Why did I choose this image?

I picked this photo as it is very nostalgic for me. I grew up in North Carolina, with my grandparents living in Carteret County, miles from the lighthouse. I grew up going to this lighthouse at least once a year, as well as exploring the island it is on. Lighthouses have always sparked my interest, with the history of each being so unique and each being shaped and created in its own way.
